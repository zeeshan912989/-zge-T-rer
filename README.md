# Özge Törer - Luxury Fan Photo Album

A premium, interactive, and responsive photo album website dedicated to Turkish actress **Özge Törer** (famous for her role as Bala Hatun in *Kuruluş: Osman*).

## Features
- **Luxury Aesthetics:** Dark theme with gold accents, smooth hover animations, and premium typography.
- **Visual Masonry Grid:** Displays 58 high-definition photos in their native aspect ratios using a fluid CSS Columns masonry layout.
- **Lightbox Viewer:** Text-free, full-screen interactive slide viewer with touch swipe controls, auto slideshow progress, download, and fullscreen capabilities.
- **Pure Frontend:** Built entirely using semantic HTML5, custom vanilla CSS variables, and modern ES6+ Javascript (no frameworks).
- **Responsive & Optimized:** Runs smoothly across mobile, tablet, and desktop viewports.

## File Structure
- `index.html` - Semantic structure.
- `style.css` - Main styles, CSS grid/masonry, and responsiveness.
- `script.js` - Data store (58 image entries), Lightbox events, autoplay logic, and scroll states.
- `assets/images/` - Portfolio images.
